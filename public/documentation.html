Movies Endpoints

Get All Movies

This endpoint returns a full list of movies in the database.

  • URL: /movies
  • Method: GET
  • Query Parameters: N/A
  • Expected Body Format: N/A
  • Response Format: JSON

Get Movie by Title

This endpoint returns details about a given movie by title.

  • URL: /movies/:Title
  • Method: GET
  • Query Parameters: Title (String)
  • Expected Body Format: N/A
  • Response Format: JSON

Get Movies by a Given Director

This endpoint returns a list of all movies by a given director, identified by the director's name.

  • URL: /movies/Director/:Name
  • Method: GET
  • Query Parameters: Name (String)
  • Expected Body Format: N/A
  • Response Format: JSON

Users Endpoints

Get User Information

This endpoint gets information about a given user, identified by username.

  • URL: /users/:Username
  • Method: GET
  • Query Parameters: Username (String)
  • Expected Body Format: N/A
  • Response Format: JSON

Add New User

This endpoint adds a new user.

  • URL: /users/:Username
  • Method: POST
  • Query Parameters: Name (String)
  • Expected Body Format: JSON
  • Body Parameters:
  • Username (String)
  • Password (String)
  • Email (String)
  • Birthday (Date)
  • Response Format: JSON

Update User Details

This endpoint updates details about an existing user, identified by name.

  • URL: /users/:name
  • Method: PUT
  • Query Parameters: Name (String)
  • Expected Body Format: JSON
  • Body Parameters (Provide at least one):
  • Username (String)
  • Password (String)
  • Email (String)
  • Birthday (Date)
  • Response Format: JSON

Add Movie to a User's Favorites

This endpoint adds a new title to a user's list of favorite movies.

  • URL: /users/:Username/movies/:MovieID
  • Method: PUT
  • Query Parameters: Username (String); MovieID (Unique ID))
  • Expected Body Format: N/A
  • Response Format: JSON

Remove Movie from a User's Favorites

This endpoint removes a title from a user's list of favorite movies.

  • URL: /users/:Username/movies/:MovieID
  • Method: PUT
  • Query Parameters: Username (String); MovieID (Unique ID))
  • Expected Body Format: N/A
  • Response Format: JSON

Delete a User

This endpoint deletes an existing user, identified by Username.

  • URL: /users/:Username
  • Method: DELETE
  • Query Parameters: Username (String)
  • Expected Body Format: N/A
  • Response Format: JSON
